# Audit AI - Frontend Website

Website penjualan jasa audit menggunakan AI dengan sistem pembayaran terintegrasi.

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript
- Lucide React (Icons)
- Recharts (Visualisasi)

## Fitur

- Landing Page dengan hero section dan value proposition
- Pricing page dengan 3 paket
- Order form dengan validasi
- Payment page dengan multiple metode pembayaran
- Dashboard untuk memulai audit
- Audit result page dengan visualisasi
- Payment status tracking
- Responsive design

## Instalasi

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Struktur Halaman

1. `/` - Landing Page
2. `/pricing` - Pricing Page
3. `/order` - Order Form
4. `/payment` - Payment Page
5. `/pending` - Waiting/Pending Page
6. `/dashboard` - Dashboard (setelah bayar)
7. `/result` - Audit Result Page
